Essence

Essence is a lightweight CSS boilerplate meant to help you build visually appealing, responsive web apps. By design, it provides you with only what you need without the bloat.

Star Follow @chrismgonzalez

Get up and running

Use one of the two following options for setup and installation

CSS (Simple)

Download the stylesheet, link it in the head, and place all custom CSS in a separate stylesheet

Sass (recommended)

Download the source, and setup a Sass workflow with npm scripts
